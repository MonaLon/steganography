## Installation

You will need Python3 to run these files.

Install dependencies.

```bash
pip3 install -r requirements.txt
```

You also will need the C library called Enchant. Run:

```bash
brew install enchant
```

## Running

```bash
python3 detect.py
```

or

```bash
python3 main.py
```
